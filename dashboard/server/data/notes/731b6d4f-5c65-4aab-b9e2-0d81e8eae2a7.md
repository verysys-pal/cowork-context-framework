**1. 원격 서버에서 seCall 실행**
```bash
secall serve --port 8080
```
이 서버가 Obsidian 플러그인이 붙을 대상입니다.

**2. Obsidian이 볼트 파일을 볼 수 있게 만들기**  
원격 SSH 환경이면 로컬 Obsidian에서 바로 원격 경로를 못 여는 경우가 많아서, 아래 중 하나를 씁니다.

- `sshfs` 마운트
- `rsync` 동기화
- `git` 동기화

가장 간단한 건 `sshfs`입니다.

```bash
mkdir -p ~/mnt/seCall
~/opt/sshfs/usr/bin/sshfs -p 22251 mhdev@141.223.48.218:/home/mhdev/obsidian-vault/seCall ~/mnt/seCall
```

그다음 Obsidian에서 `~/mnt/seCall`을 Vault로 엽니다.

**3. Obsidian에서 플러그인 활성화**
- `Settings` → `Community plugins`
- `Restricted mode` 끄기
- `seCall` 활성화
- Server URL을 `http://127.0.0.1:8080`으로 두기

**4. 실제 사용**
- `seCall Search`: 대화 세션 검색
- `Daily Note`: 날짜별 작업 정리
- `Graph Explorer`: 세션 관계 탐색
- 상태바: 서버 연결 및 세션 수 확인

**5. 자주 보는 경로**
- 볼트: `/home/mhdev/obsidian-vault/seCall`
- 플러그인: `/home/mhdev/obsidian-vault/seCall/.obsidian/plugins/obsidian-secall/`
- API 상태 확인: `http://127.0.0.1:8080/api/status`
