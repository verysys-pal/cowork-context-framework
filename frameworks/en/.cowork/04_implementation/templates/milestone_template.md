## Template Usage Rules

- If there are no related Tasks yet, do not keep placeholders. Leave an empty list or write `None`.
- For `Status` and `Main Phase`, keep only the one real selected value and remove the example set and comments.
- Remove `<!-- -->` comments and unused check items after the draft is approved.
