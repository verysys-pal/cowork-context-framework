# Tech Stack Registry

> Registry of the technologies used in this project and the ADR basis for each choice

---

## Purpose

Register the project's languages, frameworks, libraries, communication protocols, and infrastructure in one place,
and link the ADR that explains each decision.

- When a technology is selected, an ADR is automatically created and then registered here.
- `coding_convention.md` is dynamically composed based on the technologies registered in this document.
- When needed, the tech-stack fields in the four AI entrypoints are also reviewed against this document and `tooling_environment_guide.md`.
