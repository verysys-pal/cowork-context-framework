# Architecture Decision Record (ADR) Template

> 아키텍처 의사결정 기록 — 왜 이 결정을 했는가를 미래의 우리에게 남기는 문서

---

## Template 사용 규칙

- 이 파일은 복사 원본이다. 실제 ADR 문서를 만들 때는 `ADR-000`, `INT-000`, `MS-000` 등을 실제 ID로 교체한다.
- `Requirement / Story`는 실제 연결된 `FR-*`, `NFR-*`, `US-*`만 남기고, 없는 관계는 `없음` 또는 해당 없음으로 정리한다.
- 검토하지 않은 옵션 섹션은 그대로 두지 말고 삭제하거나 실제 옵션으로 교체한다.
- `<!-- -->` 가이드 주석은 승인 전 초안 작성용이며, ADR을 registry에 등록할 때는 제거한다.
- 아래 `ADR 필요성 판단`은 모든 설계 메모를 ADR로 승격하지 않기 위한 최소 체크다. 해당하지 않으면 관련 canonical 문서나 세션 로그에만 기록한다.
