# Domain Glossary

> 도메인 용어집 — 프로젝트의 공통 언어를 정의한다

---

## 문서 정보

| 항목 | 내용 |
| --- | --- |
| 관련 Intent | 활성 Intent 또는 관련 `INT-*` 참조 |
| 버전 | 프로젝트 기준 |

---

## 목적

이해관계자 전원(Human, AI, 고객)이 **동일한 용어를 동일한 의미로** 사용하기 위한 기준 문서.
모든 산출물(요구사항, 기능 명세, 코드, 테스트)에서 이 용어집의 정의를 따른다.

---

## 작성 기준

- 이 문서에서는 예시 주석 대신 실제 등록값 또는 `현재 등록 ... 없음` 행을 사용한다.
- 용어, 약어, 규칙이 늘어나면 빈 예시를 복제하지 말고 실제 항목만 추가한다.
