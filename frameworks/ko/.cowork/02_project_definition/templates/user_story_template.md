# User Story Template

> 사용자 관점의 기능 명세 — Human과 AI가 "무엇을 만들 것인가"에 대해 합의하는 계약

---

## Template 사용 규칙

- 이 파일은 복사 원본이다. 실제 Story 문서를 만들 때는 `US-000`, `INT-000`, `MS-000`을 실제 ID로 교체한다.
- 관련 Milestone이 없으면 placeholder를 남기지 말고 `없음`으로 쓰거나 해당 줄을 정리한다.
- `<!-- -->` 주석과 예시 Gherkin 문구는 초안 작성 후 제거한다.
- Acceptance Criteria 번호와 우선순위, 크기 추정은 실제 합의값만 남긴다.
