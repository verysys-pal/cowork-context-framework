# Intent Template

---

## Template 사용 규칙

- 이 파일은 복사 원본이다. 실제 Intent 문서를 만들 때는 `INT-000`을 실제 `INT-*`로 교체한다.
- `<!-- -->` 가이드 주석은 초안 작성용이며, instance 문서로 승격할 때 제거한다.
- 관계가 없는 항목은 placeholder를 유지하지 말고 비우거나 `없음`으로 정리한다.
- 사용하지 않는 섹션이나 체크 항목은 그대로 두지 말고 삭제하거나 실제 내용으로 바꾼다.
