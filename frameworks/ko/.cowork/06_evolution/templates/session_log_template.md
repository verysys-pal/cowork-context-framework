# Session Log Template

> 세션 기록 — AI-Human 협업 세션의 활동과 결과를 추적

---

## Template 사용 규칙

- 이 파일은 새 세션 로그 생성용 템플릿이다.
- `INT-000`, `MS-000`, `TASK-000`, `#000`은 실제 세션 값으로 교체한다.
- 관련 항목이 없으면 placeholder를 남기지 말고 `없음`으로 적는다.
- 빈 체크 항목과 빈 표 행은 실제 기록 후 정리하고, 세션 종료 시 핵심 정보만 남긴다.
- migration note, rollout memo, integration memo 같은 경계형 메모는 별도 임시 문서를 만들기보다 이 로그에 먼저 기록하고, 반복 참조되거나 승인된 기준이 되면 관련 source 문서로 승격한다.
- 승격은 기본값이 아니다. 아래 `승격 후보 판단`에 분명한 근거가 있을 때만 검토하고, 아니면 세션 로그에 유지한다.
