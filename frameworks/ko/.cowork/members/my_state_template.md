---

## Template 사용 규칙

- 이 파일은 `members/<이름>/workspace/my_state.md` 생성용 템플릿이다.
- `Task ID`, `관련 Milestone`, `참조 세션 로그`에는 실제 값만 적고, 더미 ID를 남기지 않는다.
- 담당 작업이나 이월 항목이 없으면 빈칸 대신 `없음`으로 정리한다.
- 세션 Intent 표의 결과는 실제 세션 종료 상태로 갱신한다.
- 1인 프로젝트에서는 `이름`에 확정된 사용자 식별자를 넣고, `역할 (Role ID)`는 `프로젝트 오너`, `활성 Milestone`은 `project_state.md` 기준 현재 활성 Milestone 또는 `없음`으로 시작한다.
- 1인 프로젝트에서 아직 개별 Task가 없으면 `현재 담당 작업`은 `없음`, 세션 Intent의 대상 Task는 `프로젝트 공통`으로 둔다.
- `다음 시작점`은 최초 생성 시 `project_state.md`의 현재 다음 시작점을 요약 복사하고, `참조 세션 로그`는 최신 개인 세션 로그를 적는다.

---

## 목적

`project_state.md`가 프로젝트 전체의 상태 인덱스라면,
이 문서는 **나의 담당 작업 상태**를 요약한다.

- 세션 시작 시 AI가 이 문서를 읽어 내 작업 맥락을 복원한다
- 팀원도 이 문서를 참조하여 내 진행 상황을 파악할 수 있다
- 개인 세션 로그의 상세 기록을 대체하지 않는다
